# AI Assistant Development

## Project Overview
This project is a basic AI Assistant developed using Python as part of the AI & Prompt Engineering Internship.

## Features
- Answer Questions
- Summarize Text
- Generate Creative Content
- User-Friendly Menu
- Feedback Collection System
- Saves Feedback to a Text File

## Technologies Used
- Python

## Files Included
- ai_assistant.py
- feedback.txt

## How to Run
1. Open the project in VS Code.
2. Run ai_assistant.py.
3. Select an option from the menu.
4. Interact with the AI Assistant.
